Welcome to RUNNING Excerises

Rules:

  1. Run during daylight hours, if possible.

  2. Stay on the sidewalk or shoulder of a road, if possible.

  3. Run facing oncoming cars.

  4. Always yield to vehicles at intersections.

  5. Obey all traffic rules and signals.

  6. Only run through neighborhoods and parks and on trails known to be safe.

  7. Dress for the weather.
